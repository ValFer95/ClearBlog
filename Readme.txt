
****************************************************************************************************************
Objectif de ClearBlog :

****************************************************************************************************************

	ClearBLog est un blog créé pour m'entraîner et concrétiser les concepts Django et Bootstrap appris le 
	long de mon auto-formation.
	Il est uniquement voué à l'exercice et à la démonstration !



****************************************************************************************************************
Usercases back-end (pour l'administrateur du blog) :

****************************************************************************************************************

	Gérer les catégories (créer, modifier, supprimer)
	Gérer les articles du blog (créer, modifier, publier, supprimer)
	Gérer les commentaires du blog (modérer, supprimer)


****************************************************************************************************************
Usercases front-end (pour l'internaute) :
****************************************************************************************************************

	Afficher l'ensemble des articles publiés
	Filtrer les articles par catégorie
	Savoir le nombre d'articles publiés dans chaque catégorie
	Consulter l'article complet
	Prendre connaissance des commentaires des internautes
	Laisser un commentaire


****************************************************************************************************************
Environnement technique de développement :
****************************************************************************************************************

	Framework 				--> Django 3.2

	Style 					--> Bootstrap 5.0.0

	Application Django utilisée		--> django-crispy-forms 1.11.2

	Environnement de développement		--> PyCharm Community 2020.3.5

	Gestionnaire de paquets			--> pip 21.1

	Gestionnaire de sources			--> Git version 2.31.0.windows.1
